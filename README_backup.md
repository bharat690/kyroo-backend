# KIRO — AI Life Operating System

Your AI best friend who runs your whole life.
Fitness. Money. Mind. Sleep. On WhatsApp. Every day.

## Stack
- Frontend: Next.js + Tailwind
- Backend: FastAPI + PostgreSQL
- AI: Claude API + LangGraph
- Delivery: WhatsApp API
- Hosting: Render

## Structure
- kiro-frontend/ — Next.js website
- kiro-backend/  — FastAPI backend (coming soon)

## Local Development
cd kiro-frontend && npm run dev
